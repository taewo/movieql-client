5. graphql-tag는 우리가 프론트엔드에 graphql을 작성하는 방법.
    https://github.com/apollographql/graphql-tag에서 사용법 확인

    import gql from 'graphql-tag';

    const query = gql`
    {
        user(id: 5) {
        firstName
        lastName
        }
    }
    `

    이런 문법으로 쓰인다.
